# 大模型应用平台 介绍

本平台基于 Streamlit 与 DeepSeek 构建，提供四个核心功能页面。

## 智能对话
支持多轮对话，可选择 deepseek-chat 与 deepseek-reasoner 两种模型，
并调节生成温度（temperature）控制回答的随机性。

## 知识库问答（RAG）
用户上传 txt / md / pdf 文档后，系统将文本切分为若干片段，
使用 TF-IDF 向量建立索引。提问时检索最相关的片段并交给大模型生成答案。

## 表格数据分析
上传 CSV 文件即可查看字段类型、统计摘要与分布图，
还能用自然语言向模型提问，例如“平均价格是多少”。

## 设置
集中管理 DeepSeek 的 API Key、Base URL 与模型名称，并支持一键连接测试。
